# Changelog

All notable changes to Theme Maker Y1 are documented here.

## [1.0.0] - 2026-03-29

- feat: Added Pink, Light Yellow and Purple device color variants
- feat: Font replacement: replace the theme font directly from the properties sidebar
- improve: Redesigned ClickWheel with velocity-based scrolling, adaptive thresholds and post-release momentum
- improve: ClickWheel buttons now show press feedback with icon color change and scale animation
- improve: Center button is now the same solid color as the device body with subtle color-only press feedback
- improve: Device background colors are now more matte and visually distinct
- improve: Global text size audit — all UI text is now appropriately sized and legible
- fix: Dialog text is now automatically readable on any background color using perceived luminance detection
- remove: Removed Google Generative AI dependency — AI icon editor now uses OpenAI

## [0.9.0] - 2026-03-10

- feat: AI icon editor: generate and paste icons using an AI image model
- improve: Image optimization pipeline — exported themes have smaller file sizes
- improve: Image mask dimensions corrected for accurate overlay rendering
- improve: Icon sizes for Settings view items are now accurately represented

## [0.8.0] - 2026-02-20

- feat: Expanded Settings view simulator: Brightness, Equalizer, Date & Time, Language, Wallpaper and About screens
- improve: Settings menu navigation is more accurate to the real device behavior
- fix: Home screen dimensions now match real device proportions

## [0.7.0] - 2026-02-05

- feat: Interactive tutorial on first launch — walks through navigation, copy/paste and export
- feat: Show Tutorial option added to the Help menu
- feat: Download theme directly from the Gallery
- fix: Backlight simulation now toggles correctly

## [0.6.0] - 2026-01-20

- feat: Links to the Timmkoo Modders Discord and r/innioasis Reddit community
- feat: Theme Gallery: browse and clone community themes
- fix: ClickWheel scroll direction and sensitivity improvements
- improve: Device shell metallic finish with per-color gradients and shine overlays
